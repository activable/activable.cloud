## Key decisions

1. **v1 scope pivot — substrate-first.** v1 ships the knowledge graph foundation (Postgres+AGE + Rust query API + Go GraphQL API server + AWS ingestion). Cloud attack-graph (Parliament IAM eval + agentic purple-team) demoted to v2 enablement #1. K8s ingestion deferred to v1.5. Estimated v1 effort dropped from 14-20 wks to 6-9 wks.

2. **SaaS platform, not CLI binary.** Interface = GraphQL API (read queries + ingestion-trigger mutations). No CLI in v1. Development + delivery = Kubernetes manifests (Kind/k3d + Helm chart). Docker Compose retained only for Postgres+AGE fallback.

3. **GraphQL API surface.** Queries: findNode, walkEdges, pathFinder, blastRadius, subgraph. Mutations: triggerIngest(provider, regions) -> async job -> ingestStatus(runId).

4. **Partial-failure ingestion.** Per-service commits + ingest_run metadata table with UUID run ID. Enumerate all enabled regions by default.

## Process lessons

**"The bench IS the test" trip-wire.** The session's first corrective action was surfacing that PR #2 shipped with zero unit tests. The orchestrator's pre-commit enumeration rationalized "the bench IS the test" — a named trip-wire in the project rules. Root cause: completion pressure overrode mandatory tester subagent dispatch. Corrective: 84 tests added retroactively; new Testing & Coverage HARD RULE (98% branch default) installed in `.claude/rules/development-rules.md`.

**Plan-taxonomy exposure.** Five commits + PR title/body shipped with "Slice A Phase 2" in public artifacts — violating the No Plan References rule. Required a full interactive rebase + force-push + PR body rewrite. Corrective: new Plan-Taxonomy Exposure HARD RULE installed with forbidden-token list, enforcement-grep recipe, pre-commit hook (`scripts/check-no-plan-taxonomy.sh`).

**Scope pivots mid-session are OK if captured.** Three scope corrections happened during this session (substrate-first, SaaS not CLI, GraphQL not CLI). Each was captured in the brainstorm summary, reflected in the plan files, and propagated to ROADMAP. The plan survived because the persistent artifacts (brainstorm report, phase files, ROADMAP session notes) tracked every pivot.
